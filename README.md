## Next.js App Router Course - Starter

This is the starter template for the Next.js **App Router Course**. 

For more information, see the [course curriculum](https://nextjs.org/learn) on the Next.js Website.

### Technologies
- Next.js
- Typescript
- Tailwind
- Next-auth
- PostgreSQL
